# Demo Video Script (1-2 minutes)

## Introduction (10 seconds)
"Hi! This is my Real Estate Analysis Chatbot built with React and Django. It analyzes real estate data from Excel files and provides insights through natural language."

## Show the Interface (10 seconds)
- Open the application in browser
- Show the clean, modern UI with chat interface
- Point out the file upload section

## Upload Data (15 seconds)
- "First, I can upload an Excel file with real estate data"
- Click upload button, select file
- Show success message

## Demo Query 1 - Single Area Analysis (20 seconds)
- Type: "Give me analysis of Wakad"
- Show the response:
  - Natural language summary
  - Price trend chart
  - Filtered data table
- "As you can see, it provides a comprehensive summary, visualizes trends, and shows the raw data"

## Demo Query 2 - Comparison (20 seconds)
- Type: "Compare Ambegaon Budruk and Aundh demand trends"
- Show the comparison:
  - Side-by-side analysis
  - Multi-line chart comparing both areas
  - Data table with both areas

## Demo Query 3 - Time-based Analysis (15 seconds)
- Type: "Show price growth for Akurdi over the last 3 years"
- Show filtered results for specific time period
- Highlight the chart showing growth trend

## Show Additional Features (10 seconds)
- Click "Download CSV" button
- Show the downloaded file
- "Users can export the filtered data for further analysis"

## Technical Highlights (10 seconds)
- "The backend uses Django REST Framework with pandas for data processing"
- "Frontend is built with React, Bootstrap, and Chart.js"
- "It intelligently parses queries to extract areas and time periods"

## Closing (10 seconds)
- "The application is fully responsive, deployed, and ready to use"
- "Check out the GitHub repository for the complete code and documentation"
- "Thank you!"

## Recording Tips

1. Use screen recording software (OBS, Loom, or QuickTime)
2. Record in 1080p resolution
3. Speak clearly and at moderate pace
4. Show mouse cursor for better clarity
5. Keep browser window maximized
6. Close unnecessary tabs/applications
7. Test audio before recording
8. Do a practice run first

## What to Show on Screen

- Clean browser window with the application
- Terminal windows (optional, briefly)
- Sample Excel file structure (optional)
- GitHub repository page at the end

## Video Editing (Optional)

- Add text overlays for key features
- Speed up file upload/loading sections
- Add background music (low volume)
- Include intro/outro slides with your name and project title
